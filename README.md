| Technology        | Used                             |
| ----------------- | -------------------------------- |
| Templating Engine | Microsoft's Razor parsing engine |
| ORM               | Entity Framework                 |
| Database          | mysql(XAMPP) through Pomelo      |
| Authentication    | Identity                         |
